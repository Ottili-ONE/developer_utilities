# Ottili Developer Utilities

Ottili Developer Utilities is a small public utility API operated by Ottili ONE.

It is intentionally limited to safe, cheap v0.1 endpoints: time/date, calculators, unit conversion, text tools, encoding, hashing, UUID/random generation, JSON utilities, QR codes, weather with caching, and debug helpers.

## Service

- Official instance: https://utils.ottili.one
- Service code: `services/developer_utilities/`
- API docs: `/docs`

## Limits

- No API key: 1,000 requests / day / IP
- Free API key: 100,000 requests / month
