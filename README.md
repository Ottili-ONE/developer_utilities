# developer_utilities

Ottili Developer Utilities is the public developer utility API for `utils.ottili.one`.

- Free utility APIs for everyday developer tasks
- No signup required for basic usage
- Free API keys unlock higher limits
- Apache-2.0 source code, with Ottili branding reserved
- Official hosted instance: `https://utils.ottili.one`
- Self-host instructions: `services/developer_utilities/README.md`

Service docs live in `services/developer_utilities/README.md`.
